Essential Calculus 2nd Edition - James Stewart:

Chapter 1 - Functions and Limits:

1.1 Functions and Their Representations

1 If \(f(x) = x + \sqrt{2-x}\) and \(g(u) = u + \sqrt{2-u}\), is it true that f = g?

Solution:

Yes. Since both functions have the same domain and image, we can set u equal to x and write \(g(u) = g(x)\), so \(g(x) = x + \sqrt{2-x}\).
